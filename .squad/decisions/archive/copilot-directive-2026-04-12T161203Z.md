### 2026-04-12T16:12:03Z: User product direction for campaign management and note UX
**By:** FFMikha (via Copilot)
**What:**
- Linked guest memberships should display differently than unlinked guest memberships in the member list.
- The post-linkage indicator explaining that note history was preserved should only appear immediately after linkage, not persist afterward.
- Linked guest accounts should behave more like real signed-in accounts in navigation, including campaign switching plus New campaign and Sign out actions, while non-owners still cannot modify campaign settings.
- Campaign owners should be able to consolidate note authorship from duplicate members onto a single member without rewriting full note history.
- The app needs search capabilities.
- Notes should support friendlier rich-text formatting and render saved formatting appropriately.
- Notes should be browsable in a folder-like way through flexible graph-style links between tags, not rigid slash-delimited parent/child paths.
- The note list/detail layout should be reworked to be more mobile-friendly and use screen real estate better.
**Why:** User request — captured for team memory
